# MAD GCP Automation

This repository contains a collection of GCP automation projects. Each folder is an independent product automation.

## Projects

### [budget-deployer](./budget-deployer)

**GCP Project Budget Enforcement Automation**

This repository provides a robust and automated solution to enforce a hard budget limit on Google Cloud Platform (GCP) projects. It prevents unexpected costs by automatically disabling billing for a project when its spending exceeds a predefined budget.

### [dataset-deployer](./dataset-deployer)

**GCP Dataset Deployer**

This project provides automation for deploying and managing datasets in Google Cloud Platform, likely targeting BigQuery.
